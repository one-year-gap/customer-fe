### Front-end
- **Next.js**: 라우팅 관리 편의성 및 서버 사이드 렌더링을 통한 초기 로딩 속도 최적화.
- **Tailwind CSS**: 유틸리티 우선 방식으로 생산성을 높이고 일관된 UI 구현.
- **TypeScript**: 엄격한 타입 정의를 통해 협업 시 버그를 사전에 방지하고 데이터 형식을 통일.
- **TanStack Query**: 캐싱 및 자동 재요청 기능을 활용하여 네트워크 자원 낭비 최소화 및 데이터 동기화 효율화.
- **Zustand**: 직관적이고 가벼운 상태 관리를 통한 데이터 흐름 제어.
- **pnpm**: 고성능 패키지 매니저를 사용하여 의존성 관리 및 빌드 속도 개선.

### Tools
- **Github**과 **Jira**를 연동하여 작업 단위를 이슈별로 관리.
- `feat`, `fix`, `refactor`, `test`, `config` 등 명확한 브랜치 전략 기반의 PR 워크플로우 운영.
- **Husky**를 통한 커밋 컨벤션 준수, **ESLint** & **Prettier**를 이용한 코드 일관성 유지, **Code Rabbit**을 활용한 AI 코드 리뷰 도입.
- **Figma**를 활용하여 디자인 및 UI 작업을 공유하고 동시 협업 진행.
